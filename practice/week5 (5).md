# 5주차 실습 기록

## 선택한 템플릿 및 주제

Pygame을 활용한 Sprite Basics 템플릿을 기반으로, 고양이 이미지를
캐릭터로 사용하여 화면에서 이동하고 소리를 재생하는 간단한 게임을
구현하였다. 방향키를 이용해 캐릭터를 움직이고, 스페이스바를 누르면
고양이 소리가 나도록 구성하였다.

------------------------------------------------------------------------

## 사용한 AI 프롬프트 (요약)

-   이미지 파일을 pygame에서 불러오는 방법\
-   이미지 배경을 투명하게 만드는 방법\
-   캐릭터를 이동시키는 코드 작성 방법\
-   화면 밖으로 나갔을 때 반대편에서 나오게 하는 방법\
-   키보드 입력 처리 방법\
-   스페이스바를 눌렀을 때 소리가 나게 하는 방법\
-   pygame에서 사운드 파일을 불러오는 방법\
-   오류(이미지/사운드 로드 실패, 응답 없음) 해결 방법

------------------------------------------------------------------------

## AI 답변에서 도움이 된 것

AI를 통해 pygame에서 이미지와 사운드를 불러오는 기본적인 방법을 이해할
수 있었다. 특히 `convert_alpha()`를 사용해 투명 배경을 유지하는 방법과,
`pygame.key.get_pressed()`를 이용한 키 입력 처리 방식이 도움이 되었다.
또한 사운드를 추가하기 위해 `pygame.mixer`를 사용하는 방법과, 이벤트
처리 구조를 이해하는 데 큰 도움이 되었다.

------------------------------------------------------------------------

## AI 답변을 수정하거나 버린 것

초기에 제시된 코드에서는 이미지와 사운드 파일의 경로가 맞지 않아
실행되지 않는 문제가 있었다. 이를 해결하기 위해 파일을 같은 폴더에
위치시키거나 절대 경로를 사용하는 방식으로 수정하였다. 또한 사운드
초기화 과정에서 오류가 발생하여 try-except 구문을 활용해 오류를
방지하도록 코드를 수정하였다. 일부 불필요하거나 복잡한 부분은 제거하고,
실행이 안정적으로 되도록 코드를 간단하게 정리하였다.
